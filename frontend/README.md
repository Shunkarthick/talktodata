# TalkToData Frontend

React + TypeScript + Vite frontend for TalkToData.

## Setup

```bash
npm install
npm run dev
```

## Tech Stack

- React 18
- TypeScript
- Vite
- TailwindCSS
- Apache ECharts
- TanStack Query
- Zustand (state management)
- React Router

## Development

Frontend will be available at: http://localhost:5173

Backend API: http://localhost:8000
